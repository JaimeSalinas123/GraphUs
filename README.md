# 🎨 GraphUs

¡Bienvenido a **GraphUs**! Esta herramienta es tu mejor aliada para diseñar diagramas Entidad-Relación (ER) de una forma rápida, intuitiva y profesional. 

Ya sea que necesites documentar un proyecto universitario o estructurar una base de datos compleja, GraphUs te permite construir y modificar tus diagramas sin complicaciones, manteniendo siempre un diseño limpio y ordenado.

---

## ✨ Características principales

*   **Diseño intuitivo:** Crea y ajusta tus diagramas con fluidez para que se adapten exactamente a lo que necesitas.
*   **Integración de código:** Importa y exporta tus modelos de forma sencilla, permitiendo que tu base de datos y tu diseño siempre estén en sintonía.
*   **Exportación profesional:** Genera imágenes y documentos en PDF con alta calidad visual, ideales para presentaciones o documentación técnica.
