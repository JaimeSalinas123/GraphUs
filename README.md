#GraphUs
Este proyecto consiste en el desarrollo de una herramienta diseñada para crear y gestionar diagramas, con especial énfasis en los diagramas Entidad-Relación (ER).
La aplicación permite construir y modificar diagramas de forma intuitiva, personalizando su estructura según las necesidades del usuario.

Además, el sistema incorpora funciones para exportar e importar código basado en los modelos generados, facilitando su integración en otros entornos de trabajo.
También ofrece la posibilidad de generar imágenes de los diagramas y exportarlas en formato PDF, manteniendo una excelente calidad visual para fines de documentación, presentación o resguardo.
